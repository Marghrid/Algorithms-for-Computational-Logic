wmifumax-0.9
============
wmifumax is a solver for weighted (partial) MaxSAT.
It implements the  algorithm by Manquinho et al. [MSP09]  using
minisat 2.2 [ES03] as the underlying solver.

Invoke by ./wmifumax INSTANCE, or ./wmifumax - to read from standard input.

More information can be found at http://sat.inesc-id.pt/~mikolas/sw/mifumax/
